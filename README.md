# dotCMS Doc Converter 

This plugin uses the Gotenberg docker image to transform one document type into another.

https://gotenberg.dev/





## Use Case:
Convert a .docx into html
Convert a webpage into a PDF
etc.


## App Config



## Building

Build using maven
```
./mvnw clean install
```


To skip tests, run

```
./mvnw clean install -DskipTests
```
